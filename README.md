# Lox Tree-walk Interpreter in Rust

A Rust implementation of the Lox programming language from **Crafting Interpreters** by Robert Nystrom.

This project follows the book at https://craftinginterpreters.com/

## Building

```bash
cargo build --release
```

## Running

Execute Lox source files:
```bash
cargo run file.lox
```

Or start a REPL:
```bash
cargo run
```
